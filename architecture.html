3
Post-Enforcement Analysis
After enforcement, the verdict is checked against behavioral patterns. Can upgrade the decision if anomalies are found.

Boundary Probing Detection

Detects agents trying slight variations of blocked actions. Triggers after N+ blocked calls with same tool but different parameters within 2 minutes.

Repetition Loop Detection

Catches agents stuck in infinite loops. Triggers when N+ identical actions (same parameter hash) occur within 2 minutes.

Chain Detection

Multi-step attack pattern recognition with intent classification (recon → escalate → exfil). Supports custom YAML chain patterns with configurable time windows.

Rate Limiter Recording

Records action fingerprints (tool + param hash + timestamp + blocked). Updates lockout state for future short-circuit checks.

4
Audit Logger
Logs the final verdict to in-memory ring buffer (1000 entries) and OpenTelemetry spans with attributes (action_id, decision, risk_level, latency_ms) and events for each rule match.
